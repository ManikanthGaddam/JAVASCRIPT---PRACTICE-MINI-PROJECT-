🚀 Mini Project: Async Data Pipeline Simulator
📌 Problem Statement

    You need to build a Node.js-based simulation of a data pipeline system that mimics how backend services handle asynchronous operations like:
            -> Fetching data from external sources
            -> Processing the data
            -> Storing it
            -> Sending it to another service

    All operations must be asynchronous and implemented using:
            -> Promises
            -> async/await
            -> (optionally) callbacks for comparison
